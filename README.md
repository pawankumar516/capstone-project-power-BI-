# capstone-project-power-BI-
This Power BI project focuses on transforming raw data into meaningful, interactive dashboards that provide clear business insights. The project demonstrates strong skills in data modeling, DAX, data visualization, and storytelling.
Objectives

Clean and transform raw data using Power Query

Build an efficient data model with relationships

Create interactive dashboards for decision-making

Analyze key KPIs, trends, and patterns

Present insights in a clear, business-friendly manner

 Tools & Technologies Used

Tool: Power BI Desktop

Data Transformation: Power Query

Calculations: DAX (Data Analysis Expressions)

Data Source: CSV / Excel / SQL Database

Visualization: Interactive charts, slicers, KPIs

Removed duplicates and null values

Renamed and standardized columns

Created calculated columns and measures

Established relationships between tables

 Data Model

Star schema design

Fact and dimension tables

Optimized relationships for performance

 Dashboards & Visuals

The report includes:

KPI Cards (Total Sales, Revenue, Profit, etc.)

Bar & Column Charts

Line Charts for trend analysis

Pie / Donut Charts for category distribution

Slicers for dynamic filtering (Date, Category, Region)

 Key Business Questions Answered

What are the overall performance trends?

Which categories or products perform best?

How does performance vary by region or time?

What factors drive higher revenue or profit?

 Key Insights

Identified top-performing categories and regions

Discovered seasonal trends and growth patterns

Highlighted underperforming areas for improvement
